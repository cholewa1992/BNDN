\chapter{Proposed System}

\section{Overview}
The system to be developed should consist of a web application (\textit{ArtShare}) and a server (\textit{ShareIt}). The system should initially support upload and download of three different media types: books, videos and music, but it should be extensible such that adding additional media types in the future can be done relatively easy.
%The purpose of the system is to handle different types of media files (e.g. books, videos, sound files or images). It must be extendible, such that new clients with similar functionality can use the same services as the client developed in this project.

%The system is aimed at anyone who wants to buy media via the internet and anyone who wants to share media they have made via the internet. This could be anyone from a young girl who wants to buy the newest music to an aspiring writer who has written a book he wants to sell.
